# 插件市场

插件市场是 Core 给管理员提供的“插件安装入口”。

管理员不需要进命令行，只需要：

- 添加插件源
- 浏览插件
- 安装 / 升级 / 卸载插件

## 一个插件源最少要提供什么

- `registry.json`
- 每个版本对应的下载地址
- 对应安装包的 `sha256`

## 一个安装包最少要有什么

- 插件二进制
- `plugin.yaml`
- 可选的 `web/` 前端资源

说明：

- `plugin.yaml` 只用于安装、发布、市场展示
- 运行时真相来自插件本身暴露的 `Info / Platform / Routes / Models / AccountTypes`

## 安装流程

1. Core 从插件源拿到下载地址
2. 下载压缩包并校验 `sha256`
3. 解压出二进制、`plugin.yaml` 和前端资源
4. 检查 `min_core_version`、`dependencies` 和插件 ID 冲突
5. 启动插件进程
6. 读取插件运行时元信息并注册路由、前端资源
7. 在管理后台显示该插件

## 升级流程

1. 下载新版本
2. 停止旧进程
3. 替换二进制和前端资源
4. 启动新版本
5. 如果失败，回滚旧版本

## 卸载流程

1. 调用插件 `Stop()`
2. 停止插件进程
3. 移除插件路由和前端资源
4. 删除插件目录和数据库记录

## 开发者发布流程

1. 按 `airgate-sdk` 规范实现插件
2. 生成 `plugin.yaml`
3. 构建二进制和前端资源
4. 打包发布
5. 把版本信息写进 `registry.json`
