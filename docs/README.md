# Core 文档

如果你第一次看 AirGate Core，只看下面 3 篇：

1. `overview.md`
2. `architecture.md`
3. `marketplace.md`

## 这 3 篇分别解决什么问题

- `overview.md`
  - Core 是什么
  - 3 个仓库怎么分
- `architecture.md`
  - Core 和插件怎么配合
  - 一个请求怎么走
  - 一个插件怎么接进来
- `marketplace.md`
  - 插件怎么安装、升级、卸载

## 其他资源

- SDK 接口定义和插件开发规范：见 `airgate-sdk` 仓库
- OpenAI 插件实现细节：见 `airgate-openai` 仓库
